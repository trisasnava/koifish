# Koifish Commands

```shell script
>  koifish  
    Start koifish in local CLI

>  koifish -h 
     -h  Help
     -v  Show version
     -s  Start a web  Koifish  in local
     -o  Run a online  Koifish in https://webassembly.sh
     -j  join our Slack Channel or WeChat Group(QR-Code)

>  koifish login  |  GitHub  account login
>  koifish user  | Get GitHub user info
>  koifish repo | Get GitHub repo info
>  koifish treding | Get GitHub treding info
>  koifish issue | get issues 
>  koifish pr | get prs 
>  etc...
```
