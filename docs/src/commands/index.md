# Koifish Commands

```shell script
> koifish -h

    koifish 0.0.1
    
    █▄▀ █▀█ ░ █▀▀ ░ █▀ █░█
    █░█ █▄█ █ █▀▀ █ ▄█ █▀█
    
    USAGE:
        koifish [OPTIONS] [SUBCOMMAND]
    
    FLAGS:
        -h, --help       Prints help information
        -V, --version    Prints version information
    
    OPTIONS:
        -j, --join <join>       Join our Slack Channel or WeChat Group(QR-Code) [default: slack]
        -o, --oline <online>    Run a online Koifish in https://webassembly.sh [default: ]
        -w, --web <web>         Start a web Koifish in local with your port [default: 2121]
    
    SUBCOMMANDS:
        help        Prints this message or the help of the given subcommand(s)
        issues      Get GitHub issues info in your repo
        login       Login to GitHub
        prs         Get GitHub prs info for your repo
        repo        Get GitHub repo info
        trending    Get GitHub trending repo info
        user        Get GitHub user info
```
